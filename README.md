# face-emotion-detiction-
Developed a deep learning–based facial emotion recognition system classifying 7 emotions from 7,066 test images. The model achieved 68.4% accuracy, 68.8% weighted precision, 68.4% recall, and 68.2% F1-score, with strongest performance on “happy” (88% F1-score) and “surprise” (79% F1-score) emotions.
